# POD GO --> Helix converter

## Description

POD_GO-Helix_converter is a desktop application designed to convert patch/preset files from the LINE6 POD GO (.pgp) format to the LINE6 Helix (.hlx) format. This allows users to transfer their custom tones and settings between the two devices. The conversion process aims to be quick and efficient, providing a seamless experience for users migrating their presets.

### Key Features:
-   **Fast Conversion:** Converts .pgp files to .hlx files almost instantly.
-   **User-Friendly Interface:** Simple and intuitive design for easy navigation.
-   **Cross-Platform Compatibility:** Built with Electron for use on Windows, macOS, and Linux.

## Usage Instructions

1.  Open the POD_GO-Helix_converter application.
2.  Click on the "Select .pgp File" button to choose the POD GO preset file you want to convert.
3.  Click the "Convert" button.
4.  Choose a location to save the converted .hlx file.
5.  Import the .hlx file into your LINE6 Helix device or Helix Native software.

### Example

Converting a .pgp file named "MyAwesomeTone.pgp":

1.  Open the application.
2.  Select "MyAwesomeTone.pgp" from your files.
3.  Click "Convert".
4.  Save the converted file as "MyAwesomeTone.hlx".
5.  Import "MyAwesomeTone.hlx" into your Helix.

## Limitations

The converter is a simple mockup and may have issues with presets containing certain effects or specific configurations.
